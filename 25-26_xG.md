# 25-26 xG Study

Updated: April 17, 2026

Method:
- Completed league matches only
- Target event: wins by 2 or more goals
- xG measure: realized match xG delta
- Winner xG delta = winner xG minus loser xG
- Practical threshold = first absolute xG-delta cutoff in the standard ladder where 2+ goal wins clear roughly 55%
- This is descriptive match-level xG, not pre-match projected xG
